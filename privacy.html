txtagent Privacy Policy

txtagent is a browser extension that rewrites text you choose to submit. This policy explains what information is processed, why it is needed, and which service providers receive it.

Text you submit

txtagent processes only the message and rewrite options you deliberately submit by pressing Rewrite. For the included free service, that information is transmitted securely to our Cloudflare Worker and then to Groq, which processes it with an AI model to generate the rewrite. Submitted messages and generated rewrites are not stored in our database.

Optional personal API key

If you add an OpenAI API key, the key is stored locally in Chrome extension storage on your device. After the free allowance is exhausted, txtagent can send your submitted text and rewrite options directly to OpenAI using that key. The key is not sent to or stored by our backend. You can remove it at any time from txtagent settings or by uninstalling the extension.

Operational information

To provide and protect the free allowance, txtagent stores an anonymous installation identifier, daily request counts, daily salted hashes derived from IP addresses, coarse country and region codes supplied by Cloudflare, browser and operating-system families, browser major version, extension version, interface language, response timing, and aggregate error counts. Daily IP hashes are used for abuse prevention and are not raw IP addresses.

Information we do not intentionally collect

txtagent does not intentionally store raw IP addresses, full user-agent strings, visited URLs, browsing history, the contents of pages, submitted messages, generated rewrites, or personal API keys on its backend. The extension runs on ordinary webpages so its floating interface is available where you write, but it does not transmit page content automatically.

Service providers

Cloudflare provides hosting, request routing, coarse location metadata, and database infrastructure. Groq processes text submitted through the included free service. OpenAI processes text only when you choose to configure and use a personal OpenAI API key. These providers process information under their own privacy terms.

Purpose and sharing

Information is used only to provide rewrites, enforce usage limits, prevent abuse, diagnose errors, and understand aggregate service reliability. We do not sell personal information, use submitted writing for advertising, or create advertising profiles. Information is shared only with the service providers described above as needed to operate txtagent or when required by law.

Retention and deletion

Submitted messages and generated rewrites are not retained by our database. Daily usage records and operational analytics are retained only as needed to operate, secure, and evaluate the service. You can clear locally stored settings by removing the extension. To request deletion of server-side records associated with an installation, contact us and include the anonymous installation identifier shown in extension diagnostics, if available.

Security and changes

Data transmitted by txtagent uses HTTPS. We may update this policy when the product or its data practices change. Material changes will be reflected here with a new effective date.

Chrome Web Store Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

Privacy questions and deletion requests can be sent through the contact address published on the txtagent Chrome Web Store listing.
