— Quick answers

Good to know.

Do I need an account or API key?+

No. Install txtagent and start with 20 free rewrites daily. A personal API key is optional.

Where does txtagent work?+

txtagent appears on ordinary websites in Chrome. Protected browser pages do not allow overlays.

Does txtagent read everything I type?+

No. txtagent sends only the text you deliberately submit when you request a rewrite.

What happens after 20 rewrites?+

The free allowance resets the next day, or you can use a personal API key.
